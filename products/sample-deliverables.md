# 納品物のサンプル

このページでは、各プランで納品されるレポートの一部をご紹介します。実際にはヒアリング内容をもとに作成されるため、より具体的で役に立つ内容になります。

## 3,000円：スタートダッシュ診断（サンプル）

【AIで削減しやすい業務の見つけ方ガイド】
AIには得意・不得意があります。まずは以下の特徴に当てはまるものからAI化を始めましょう。

**・毎回ゼロから考えている文章作成（提案文、お礼メール等）**
**・過去の類似案件で使ったフォーマットの使い回し**
**・箇条書きのメモから文章を組み立てる作業**

【そのまま使える基本プロンプト例：丁寧なお断りメール作成】
```text
あなたは優秀なビジネスアシスタントです。
以下の情報をもとに、相手の気を悪くさせない丁寧なお断りメールを作成してください。

[背景]：〇〇の提案を受けたが、現在は不要
[理由]：社内リソース不足のため
[今後の可能性]：来期以降であれば検討の余地あり
```

## 9,800円：7日間導入プログラム（サンプル）

【優先してAI化すべき業務（あなたの回答に基づく上位3つ）】
（※以下は回答例「見積作成に時間がかかる」に対する納品イメージです）
**・優先1：見積額と概算作業期間の自動計算・提案文作成**
**・優先2：初回問い合わせに対するヒアリング項目の自動返信**
**・優先3：完了後の次回案件提案のフォローメール自動生成**

【専用プロンプト例（優先1用）】
```text
あなたは[業種：Web制作]専門の優秀な見積もり作成担当です。
以下のヒアリング情報をもとに、顧客向けの見積もり提案書とその理由を記載したメール文面を作成してください。

[前提条件]
・当社の基準単価：LP制作100,000円、追加ページ30,000円/p
・納期目安：LP単体で2週間

[顧客要望]
（ここに顧客からの依頼内容を貼り付けます）

[出力形式]
1. 挨拶
2. ご要望の整理
3. 概算見積と内訳
4. 納期の見込み
5. 次のステップの案内
```

【導入初期に失敗しやすいポイントと対策】
・**失敗例**：いきなり最終決定版の長文をすべてAIに書かせようとする。
・**対策**：AIは「下書きを8割の完成度で作る優秀なアシスタント」と考えてください。まずは上記の専用プロンプトでたたき台を作り、最後の固有名詞や微細なニュアンスは人間が手直しすることで、トータルの時間は半減します。

## 19,800円：本格導入・仕組み化セット（サンプル）

（※9,800円プランの内容に加え、より高度な仕組み化ガイドが含まれます）

【AIへ渡すための社内情報整理ガイド（前提条件まとめ）】
AIの出力を劇的に安定させるには、自社の「前提条件」をまとめたドキュメント（テキストファイル）を持っておくことです。

[自社用前提条件テンプレ]
```text
# 自社情報
・会社名：株式会社〇〇
・主なサービス：[サービス名]
・強み：[他社との違い]

# 顧客対応ポリシー
・トーン＆マナー：親しみやすくも丁寧（「です・ます」調ベース）
・禁止ワード：「絶対」「必ず」等の断定表現は使わない
```
これをプロンプトの冒頭に毎回貼り付けるだけで、トーン＆マナーのブレがなくなり、外注スタッフに任せても同じクオリティの返信文が生成可能になります。

【14日〜30日ロードマップ】
・Day 1-3：優先業務トップ3のプロンプト動作テストと微修正
・Day 4-10：社内情報（前提条件）のテキスト化完了
・Day 11-20：テスト運用期間（修正箇所をマニュアルに追記）
・Day 21-30：外注・スタッフへのプロンプト共有と実践開始
