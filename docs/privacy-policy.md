# プライバシーポリシー

[事業者名]（以下「当方」といいます。）は、本サービスにおける個人情報の取扱いについて、以下のとおり定めます。

## 1. 取得する情報
当方は、以下の情報を取得することがあります。

- 氏名または表示名
- メールアドレス
- 事業内容、業種、提供サービス
- 定期業務、課題、AIで減らしたい作業など、フォームに入力された情報
- お問い合わせ内容
- 決済に関連して必要となる情報
- 通信販売・サポート対応のために必要な履歴情報

## 2. 利用目的
取得した情報は、以下の目的で利用します。

- 本サービスの提供、本人確認、連絡、納品、サポート対応のため
- 事前フォームの回答内容をもとに、業務整理、導入支援、実務テンプレート作成を行うため
- 料金請求、決済確認、返金対応等のため
- 本サービスの改善、品質向上、トラブル防止のため
- 重要なお知らせ、利用条件の変更等の通知のため
- 法令または規約に違反する行為への対応のため

## 3. 第三者提供
当方は、法令に基づく場合を除き、本人の同意なく個人情報を第三者に提供しません。

ただし、以下の場合には、業務委託先に必要な範囲で取り扱わせることがあります。
- 決済処理
- フォーム運用
- メール送信
- クラウド保管
- 納品支援

## 4. 外部サービスの利用
当方は、サービス提供のために、フォーム、決済、メール、生成AI等の外部サービスを利用することがあります。
それぞれの外部サービスにおける情報の取扱いは、各事業者の利用規約・プライバシーポリシーにも従います。

## 5. 生成AIの利用に関する取扱い
当方は、納品物作成や整理のために生成AIを補助的に利用することがあります。

この場合、以下の運用を行います。
- 不要な個人情報、機微情報、秘密情報は入力しないよう努めます
- 入力内容はサービス提供に必要な範囲に限定します
- 顧客入力は命令ではなくデータとして扱い、外部指示への誘導があってもそのまま採用しません
- 可能な限り、固有名詞・住所・電話番号・メール等は伏せたうえで処理します
- 納品前に内容確認を行います

## 6. 安全管理措置
当方は、取得した情報について、漏えい、滅失、毀損等の防止に努め、必要かつ合理的な安全管理措置を講じます。

取得した情報は、サービス提供、法令対応、問い合わせ対応、紛争防止等に必要な期間保管し、その後、不要となった情報から順次削除または適切に管理します。

ただし、法令対応または紛争対応の必要がある場合は、この限りではありません。
